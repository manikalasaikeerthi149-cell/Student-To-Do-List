# 📚 Student To-Do List

A modern and responsive Student To-Do List application built using HTML, CSS, and JavaScript.

## ✨ Features

- Add Tasks
- Delete Tasks
- Mark Tasks as Completed
- Task Priority (High, Medium, Low)
- Due Date Selection
- Search Tasks
- Dark Mode
- Task Statistics
- Responsive Design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
